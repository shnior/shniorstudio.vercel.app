# Shnior Studio

אתר התדמית הרשמי של שניאור אייזנבך — קלידן, מפיק מוזיקלי, עורך וידאו ואנימטור להפקות ואירועים.

## מבנה הפרויקט

- `index.html` — דף הבית, אזורי השירותים, קריאה לתיק עבודות ויצירת קשר.
- `portfolio.html` — תיק עבודות דינמי הנטען מ-Firebase Realtime Database.
- `admin/index.html` — מערכת ניהול תוכן מוגנת באמצעות Firebase Authentication.
- `admin.html` — הפניה ישנה אל `/admin/` כדי לא לשבור קישורים קיימים.
- `vercel.json` — הגדרות פריסה, כתובות נקיות ו-Headers בסיסיים.
- `sitemap.xml` / `robots.txt` — קבצי SEO ואינדוקס.

## ניהול תוכן

מערכת הניהול נמצאת בכתובת `/admin/` ומאפשרת לעדכן:

1. טקסט לוגו.
2. כותרת ותיאור Hero.
3. רשימת פרויקטים בתיק העבודות.
4. גיבוי וייבוא של תיק העבודות כקובץ JSON.

> חשוב: הרשאות הכתיבה בפועל חייבות להיות מוגדרות גם ב-Firebase Rules, ולא רק בממשק האתר.

## בדיקה מקומית

אפשר להריץ שרת סטטי פשוט מתוך תיקיית הפרויקט:

```bash
python3 -m http.server 4173
```

לאחר מכן לפתוח בדפדפן:

- `http://localhost:4173/`
- `http://localhost:4173/portfolio.html`
- `http://localhost:4173/admin/`

## פריסה

האתר מיועד לפריסה ב-Vercel כאתר סטטי. קובץ `vercel.json` מפעיל כתובות נקיות ומוסיף Headers בסיסיים לאבטחה ולמטמון של נכסים סטטיים.
